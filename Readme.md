# This is example simple Kafka with Golang 


### Getting start

1. Init vendor
```
make init
```
2. Build & start project
```
make build 
```
3. Send data to Kafka
```
make send 
```
4. Receive data from Kafka
```
make receive 
```
5. Stop & remove all container
```
make clear 
```
6. Other command
```
make start
make restart
make stop
make info
make exec-producer
make exec-consumer
make hello
```
